CampScrape

Reserving popular campgrounds on short notice can be often difficult if not almost impossible. With the lack of a notification or waitlist system on recreation.gov and other popular campground reservation websites, waiting for availability at a campground is a matter of refreshing the page every once in a while to check to see if you get lucky. CampScrape makes securing reservations last minute easy. The python script runs routinely on an amazon AWS server, routinely webscraping the recreation.gov site and notifying the user when a campsite becomes available.
